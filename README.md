‚ú® Key Features

AI War Chronicles: Uses Gemini 2.5 Flash to generate post-battle stories based on your performance.

Cinematic Narration: Gemini 2.5 TTS provides a deep, immersive voiceover for every victory and defeat using the "Fenrir" and "Kore" voice models.

Zero-Latency Logic: A custom "Low-HP Trigger" system that generates the AI story during the fight so it's ready the moment the match ends.

Fail-Safe Loading: Includes a robust 10-second timeout mechanism for the "Scribing the Legend" phase to ensure the game always proceeds to combat even during API latency.

Dual-Mode Controls: Full support for both Desktop (Keyboard) and Mobile (Touch Overlay).

Custom Physics Engine: Built from scratch using React and HTML5 Canvas.
